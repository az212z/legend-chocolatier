# ليجند شوكولا — Legend Chocolatier

موقع تعريفي فاخر (عربي RTL) لمحل **ليجند شوكولا** في الخبر: شوكولا فاخرة، حلويات وكيك، مشروبات، وصناديق وعلب هدايا.

- **المدينة:** الخبر — المنطقة الشرقية

## لماذا اختير
نشاط هدايا/شوكولا عالي نيّة الشراء بدون موقع (لينكتري فقط)، وصور حقيقية غنية من المحل ومنتجاته تسمح ببناء موقع راقٍ يعرض المنتجات ويحوّل الزائر إلى طلب واتساب مباشر.

## فكرة التصميم — "Modern Chocolatier"
- لوحة ألوان: **إسبريسو غامق + ذهبي دافئ + عاجي + لمسة عنابي (Burgundy)** — حسّ بوتيك عصري فاخر، مميّز عن نسخة «الصناديق» (الكاكاو/الوردي).
- خطوط: عرض **El Messiri / Reem Kufi** + متن **Tajawal** (عربي مزدوج: عنوان أنيق + متن نظيف).
- الموشن ناعم (transform/opacity فقط، يحترم prefers-reduced-motion): wisp بخار/لمعان في الهيرو، Ken-Burns بطيء، ظهور الكروت بالتدرّج، تكبير صور عند الـ hover، ولمعة ذهبية على زر الطلب.

## التقنية
HTML/CSS/JavaScript فانيلا — بدون build step، بدون `node_modules` أو `package.json`. خطوط Google عبر preconnect + display=swap. الأيقونات SVG inline (نمط Lucide). جاهز لـ GitHub Pages (يحوي `.nojekyll`).

## مصدر الصور
صور حقيقية من **خرائط قوقل** لحساب المحل نفسه، تمّت مراجعتها يدويًا (curate). تمّ **استبعاد**: لوحات الأسعار، صور بوجوه/أيدٍ مميّزة، وصور تحمل **لافتات محلات أخرى** (Chocolate Design / Chocolat Very / بستاني / Peice of Chocolate). الصور المعتمدة كلها تُظهر شعار ليجند (ميدالية الإلهة الذهبية) أو واجهة/منتجات المحل.

**المعتمدة (6):** `lg-1` (واجهة المحل/الهيرو) · `lg-8` (علب شوكولا بالشعار) · `lg-12` (صندوق حلويات بالشعار) · `lg-3` (هوت شوكليت بالشعار) · `lg-6` (الكشك مساءً بالشعار) · `lg-7` (آيس كريم).
