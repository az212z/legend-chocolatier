## 2) مخرجات ui-ux-pro-max design-system
- **Pattern:** Hero-Centric + Conversion (Hero → Features → CTA) — موسّع لأقسام النشاط (ثقة/منتجات/لماذا/معرض/طلب/موقع/CTA).
- **Colors (مُتبنّاة ومُعمّقة):** `--espresso #241712` خلفية، `--gold #C8A24A` accent، `--ivory #F6EFE3` نص، `--burgundy-bright #9B3146` لمسة، `--ink-muted #C5B49B`.
- **Typography:** بدّلت توصية Inter (لاتيني) إلى **El Messiri/Reem Kufi (عرض) + Tajawal (متن)** كما يفرض البريف للسياق العربي — أنسب وأكثر فخامة للـ RTL.
- **Effects:** أقسام واسعة (gaps 48px+)، hover ناعم، نوع عريض، 200–300ms.

## 3) قرارات UI/UX الأساسية
- **هيدر** sticky بـ backdrop-blur، شعار wordmark + ميدالية SVG، روابط smooth-scroll، CTA بارز، **برغر يفتح overlay ملء الشاشة** (100vw/100dvh خلفية إسبريسو صلبة + زر X واضح، يقفل بالـ Escape وعند اختيار رابط).
- **هيرو** بصورة الواجهة الحقيقية + scrim متدرّج لضمان التباين، عنوان بكلمة ذهبية مميّزة، CTAين (طلب + تصفّح).
- **شريط ثقة** 4 رقائق فيها التقييم الحقيقي 4.3/421.
- **منتجات**: 6 كروت بأيقونات SVG، وسم «حسب الطلب» (بلا أسعار).
- **لماذا**: تخطيط split يكسر إيقاع الشبكة (لا تكرار نمط).
- **معرض** بشبكة bento (خلايا wide/مربّعة) + lightbox-lite يعمل بالكيبورد.
- **نموذج طلب** → رسالة wa.me مُعبّأة + localStorage + توست نجاح، موسوم بوضوح كتجريبي.
- **موقع** (مدينة/أوقات/خرائط) → **CTA نهائي** → فوتر بروابط/واتساب/اتصال/خرائط/© 2026. **FABs عائمة**: واتساب + اتصال + خرائط.

## 4) سبب الألوان/الخطوط
الإسبريسو والذهبي يعكسان الشوكولا الفاخرة والإهداء، والعاجي يمنح راحة قراءة، والعنابي لمسة عمق راقية تميّز الموقع عن «الصناديق» (كاكاو/وردي). El Messiri يعطي عناوين أنيقة ذات طابع، Tajawal متن نظيف عالي الوضوح — ثنائي يخدم الفخامة والقراءة معًا.

## 5) تطبيق Hooked / UX السلوكي
Trigger (CTA «اطلب» متكرّر) → Action (نموذج قصير 3 حقول مطلوبة) → Reward (توست + رسالة واتساب جاهزة) → Investment (الطلب محفوظ في localStorage). الاحتكاك أدنى ما يمكن: زر واحد رئيسي لكل مشهد، أهداف لمس ≥48px.

## 6) Accessibility
- تباين: عاجي/مكتوم على إسبريسو ≥4.5:1؛ الذهبي للنصوص الكبيرة/الأيقونات؛ نص الزر الذهبي على إسبريسو داكن.
- HTML سيمانتيك (header/nav/main/section/footer)، تدرّج عناوين h1→h2→h3 بلا قفز.
- `skip-link`, `:focus-visible` بـ outline ذهبي 3px على كل عنصر تفاعلي.
- كل صورة alt عربي وصفي + width/height + lazy لغير الهيرو + decoding=async؛ الهيرو fetchpriority=high.
- أزرار الأيقونات لها aria-label؛ القائمة/اللايتبوكس role=dialog + aria-modal؛ التوست aria-live=polite؛ الخرائط/الأرقام dir=ltr.
- `prefers-reduced-motion`: يوقف الـ wisp/ken-burns/الريفيل ويُبقي المحتوى ظاهرًا.

## 7) iOS HIG / تفاعل
أهداف لمس ≥44–48px مع تباعد، ردّ ضغط `scale(.97)` تحت 150ms، حالة loading على زر الإرسال، الأخطاء أسفل الحقل مباشرة، لا اعتماد على hover للأساسيات.

## 8) Impeccable / Taste — اختبار القبول
- يبدو فاخرًا؟ ✓ (لقطات الهيرو/المنتجات: إسبريسو/ذهبي، ظلال متّزنة، صور حقيقية).
- سعودي مناسب للنشاط؟ ✓ نسخة عامية محايدة جندريًا.
- يقنع خلال 3 ثوانٍ؟ ✓ عنوان واضح + CTA فوري + تقييم حقيقي.
- لا يشبه قالبًا مجانيًا؟ ✓ هوية مستقلة، تخطيطات متنوّعة (لا تكرار نمط بين الأقسام).
